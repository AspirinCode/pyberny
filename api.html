berny.Berny(geom, log=None, debug=False, restart=None, maxsteps=100, verbosity=None, **params)[source]¶

Coroutine that receives energy and gradients and yields the next geometry.

Parameters:
  • geom (Gometry) – geometry to start with
  • log (Logger) – used for logging if given
  • debug (bool) – if True, the generator yields debug info on receiving the energy and gradients, otherwise it yields None
  • restart (dict) – start from a state saved from previous run using debug=True
  • maxsteps (int) – abort after maximum number of steps
  • verbosity (int) – if present and log is None, specifies the verbosity of the default Logger
  • params – parameters that override the defaults

The coroutine is to be used as follows:

optimizer = Berny(geom)
for geom in optimizer:
    # calculate energy and gradients (as N-by-3 matrix)
    debug = optimizer.send((energy, gradients))
berny.core.defaults = {'dihedral': True, 'gradientmax': 0.00045, 'gradientrms': 0.0003, 'stepmax': 0.0018, 'steprms': 0.0012, 'superweakdih': False, 'trust': 0.3}¶
  • gradientmax, gradientrms, stepmax, steprms:
    Convergence criteria in atomic units (“step” refers to the step in internal coordinates, assuming radian units for angles).
  • trust:
    Initial trust radius in atomic units. It is the maximum RMS of the quadratic step (see below).
  • dihedral:
    Form dihedral angles.
  • superweakdih:
    Form dihedral angles containing two or more noncovalent bonds.

berny.optimize(solver, geom, **kwargs)[source]¶

Optimize a geometry with respect to a solver.

Parameters:
  • solver (generator) – unprimed generator that receives geometry as a 2-tuple of a list of 2-tuples of the atom symbol and coordinate (as a 3-tuple), and of a list of lattice vectors (or None if molecule), and yields the energy and gradients (as a N-by-3 matrix or (*N*+3)-by-3 matrix in case of a crystal geometry)
  • geom (Geometry) – geometry to optimize
  • kwargs – these are handed over to Berny()

Returns the optimized geometry.

Inside the function, the solver is used as follows:

next(solver)
energy, gradients = solver.send((list(geom), geom.lattce))
energy, gradients = solver.send((list(geom), geom.lattce))

Geometry operations¶

class berny.Geometry(species, coords, lattice=None)[source]¶

Represents a single molecule or a crystal.

Parameters:
  • species (list) – list of element symbols
  • coords (list) – list of atomic coordinates in angstroms (as 3-tuples)
  • lattice (list) – list of lattice vectors (None for a moleucle)

Iterating over a geometry yields 2-tuples of symbols and coordinates. len(geom) returns the number of atoms in a geometry. The class supports format() with the same available formats as dump().

bondmatrix(scale=1.3)[source]¶

Calculates the covalent connectedness matrix.

Parameters:scale (float) – threshold for accepting a distance as a covalent bond

Returns \(b_{ij}:=R_{ij}<\text{scale}\times (R_i^\text{cov}+R_j^\text{cov})\).

inertia¶

Calculates the moment of inertia, \(I_{\alpha\beta}:= \sum_im_i\big(r_i^2\delta_{\alpha\beta}-(\mathbf r_i)_\alpha(\mathbf r_i)_\beta\big)\) where \(\mathbf r_i=\mathbf R_i-\mathbf R_\text{CMS}\).
